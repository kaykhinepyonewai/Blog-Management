
# Moon Blog

A social publishing platform that is open to all and home to diverse array of stories, ideas and perspectives like medium website

# Things that need to be done before starting the project

- Change the database password in the webconfig file based on the password of your SQL Management Studio
- If there is no rows in role table, add some rows (Admin, User, Banned) to the Role table in SQL.
- Change mailtrap's key

# Login Account
- admin@gmail.com(pw - Admin@1234)
- user@gmail.com(pw - User@1234)

## Features

- like/comment feature
- Admin & User Management / Banned Role
- Report
- archieve article lists
- related article
- search article
- Request article
- waiting list articles
- category filter article
- user lists excel file export
- can add a user by using excel import
- Trend Articles 
- password change with mail trap
- admin can change all user's password
- user can only change their own password
- admin can able to view and modify all the article lists in the dashboard


## Running MOON Blog

To run MOON Blog, first restore the script DB file provided on GitHub to your Microsoft SQL Server Management. After that, open the Moon.web.sln project and run it using the following key

```bash
 Select Ctrl+F5 to run the app without the debugger.
```
